It is important that the robot is fully functioning and that the motors are wired correctly.  To help, there is a test mode that the pupils can use to check for proper operation.

To run  the test mode, the pupils need to connect TWO wire links into the front and back of the robot. As in the following picture. The wire links are in a pack (labelled wire links) in the class kit.

When the robot is switched ON, the robot will go through a series of simple tests with the help of the team.  The details are listed in the document - “Activity 1 pupil worksheet B”.

Remember to REMOVE the two wire links.  Keep them in the kit as they can be used later on if you suspect that there is a problem with a robot.  The test can help to eliminate possible errors.
