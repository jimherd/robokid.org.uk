Notes

Batteries

The supplied rechargeable batteries are delivered in a discharged state.  You MUST charge them before the first activity.  A suitable number of chargers are supplied in the class kit.  Refer to the FAQ for details about managing the batteries.

 

Electrical safety

There are NO dangerous voltages in the equipment.  Therefore, pupils cannot get an electric shock.

 

Kit boxes

Please ask your pupils not to write or draw on the cardboard boxes.  We plan to use the same boxes when we send the kits to other schools.

 

Types of floor to be used

Robot should NOT be run on carpet or carpet like floor coverings.

 

If things go wrong

The robot cannot become dangerous.  If a pupil gets the robot into what appears to be an odd mode, then just tell them to switch off and start again.  That can cure many issues.

 

For situations where that does not solve the problem then refer to the activity notes and to the FAQ.

 

Video material

Make sure that you have tested the video material before the lesson.
